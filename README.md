# kbs-childabuse

디자인 스튜디오 Velcro에서 인턴을 하며 KBS 아동학대 아카이브 사이트 웹개발의 일부를 담당하였습니다. <br>
웹페이지에 텍스트와 각종 데이터, 디자인을 반영하는 작업과 함께 엑셀 파일을 기반으로 데이터를 시각화하고 시각화된 데이터를 편리하게 필터링하여 볼 수 있는 체크박스 등 인터페이스를 개발하고 인터랙션을 추가하는 작업을 담당하였습니다. <br>
코드는 비공개로 사이트를 업로드 합니다.
