# kbs-childabuse

디자인 스튜디오 Velcro에서 인턴을 하며 KBS 아동학대 아카이브 사이트 웹개발의 일부를 담당하였습니다. <br>
웹사이트에 텍스트와 디자인을 반영하고 엑셀 파일을 기반으로 여러가지 형태로 데이터를 시각화하고 인터랙션을 추가하는 작업을 담당하였습니다.
